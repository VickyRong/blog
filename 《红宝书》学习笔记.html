第三章 基本概念
  • 语法 
  • 数据类型
  • 控制流语句
  • 函数
  1. JS区分大小写。
  2.  标识符：  1是变量，函数，属性的名字，或者函数的参数。
                            2第一个字符不能是数字，其他字符可以由数字，下划线，美元符号组成。
  1. 关键字和保留字：可以用于表示控制语句的开始或结束，关键字和保留字都是被保留不能当作标识符的。 
  2. 基本的数据类型有六类： undefined, null ,布尔型，字符串型，数值型，对象型。
       （如果定义的变量准备在将来用于保存对象，最好将该变量初始化为null而不是其他值，null表示一个空对象指针）  
  1.  “加法”：如果只有一个操作数是字符串，则将另一个操作数转换成字符串，然后将两个字符串拼接起来。 
             var result = 5 + “5” ；// 一个数值和字符串相加
             console.log( result )  // “55”
          “减法”：如果操作数其中一个是字符串，将先用后台调用number(  )ji==将其转为数值。
             var result = 5 - “2” ;  //结果是3，因为“2”被转换成了2
          “相等”：相等和不相等－－先转换再比较，全等和不等－－仅比较不转换
               var b = NaN;
               console.log(b == NaN); //返回false NaN不等于NaN。
     7.break 和 continue 的区别
   使用break会立即退出循环，而使用continue会立即退出此次循环，然后重新进入循环，根据条件继续进行判断。
  8.typeof和instanceof的区别
     前者用来检测运算数类型（比如string，boolean等），
     后者用来检测变量是否是某个对象的实例，返回true或者false.



第四章 变量，作用域和内存问题
  • 理解基本类型和引用类型的值
  • 理解执行环境
  • 理解垃圾收集
1.作用域链的作用：保证对执行环境有权访问的所有变量和函数的有序访问。
               每个环境可以向上搜索作用域链，查询变量和函数名，不能向下查询作用域链进入内部的执行环境。
2.垃圾收集机制：找出那些不再继续使用的变量，释放其占用的内存。
       常用方式：标记清除和引用计数。
3.管理内存：
       解除引用：一旦数据不可用，最好通过将其值设置为null来释放其引用。适用于大多数全局变量和全局对象的属性。局部变量会在它们离开执行环境时自动被解除引用。
4.基本类型指的是简单的数据段，引用类型指那些可能由多个值构成的对象。     
    基本类型：
        不能给对象添加属性。
        复制变量：如果一个变量从另一个变量复制基本类型的值，会从变量对象上新建一个值，然后把该值复制到为新变量分配的位置上。
    引用类型：
        可以给对象添加属性。
                 复制变量：复制的副本上一个指针，这个指针指向存储在堆中的一个对象。




第五章 引用类型
  • 使用对象
  • 创建并操作数组
  • 理解基本的javascript类型
  • 使用基本类型和基本包装类型
1.创建Object实例的方式：var obj = new Object( )和对象字面量表示法。
2.访问对象的属性可以用点和方括号的方式访问。obj.name 和 obj[name]
          但是用方括号点好处是，当对象的属性名有关键字或者特殊字符的时候，也能访问，而点的方式不能。还有一点就是属性名可以是变量。
          obj[first name] =  “hu”; //属性名有空格也可以访问
          var person ={};
          var propertyName = “name”;
          console.log(person[propertyName]) //用propertyName代替属性名name.
3.创建Array实例的方式：使用Array构造函数和使用数组字面量。
4.数组的length属性不仅仅是可读的，他还可以设置。通过设置 obj.length = 数值；增加或减短数组长度。
5.转换方法：  valueOf( )方法返回的是数组。
                       toString( )方法会返回由数组中每个值的字符串形式拼接而成的一个以逗号分隔的字符串。
                       join( )方法可以使用不同的分隔符来构建字符串。如果不给join( )传递任何值，默认用逗号做分隔符。
  栈方法：      栈是一种后进先出的数据结构。
                        push( )方法可以接收任意数量的参数。把它们逐个添到数组末尾，并返回修改后的数组长度。
                        pop( )方法从数组的末尾移除最后一项，减少数组的length,然后返回它移除的项。     
   队列方法：   先进先出的数据结构。       
                        unshift( )方法在数组前端添加任意个项并返回新数组的长度。
   重排序方法：reverse( )和sort( )  (这两个改变原数组)
   操作方法：   concat( ),slice( )         都是将原数组复制一个副本，在副本的基础上做一系列操作形成新的数组。
                        splice( )           (改变原数组)会操作原数组，可以对原数组进行删除，插入，替换的操作。（需要提供三个参数：起始位置，要删除的项数，和要插入的项）
   位置方法：   indexOf( )和lastIndexOf( ) 两个方法都接收两个参数（要查找的项和表示查找起点位置的索引（可选））。两个方法都返回查找项在数组中的位置，找不到返回-1。
                                  indexOf( )从数组的开头向后查找。
                                  lastIndexOf( )从数组的末尾开始向前查找。
   迭代方法：   every( )      对数组中的每一项运行给定函数，如果该函数对每一项都返回true,则返回true。
                        filter( )        对数组中的每一项运行给定函数，返回函数会返回true的项组成的数组。 
                        forEach( )   对数组中的每一项运行给定函数，这个方法没有返回值。
                        map( )        对数组中的每一项运行给定函数，返回每次函数调用的结果组成的数组。
                        some( )      对数组中的每一项运行给定函数，如果该函数对某一项都返回true,则返回true。
   归并方法：   reduce( )和reduceRight( ) 两个方法都接收两个参数：一个是在每一项上调用的函数另一个是作为归并基础的初始值。
6.RegExg类型：（正则表达式）
                       g : 表示全局（global）模式,即模式将被应用于所有字符串，而非发现第一个匹配项时立即停止；
                       i  : 表示不区分大小写模式。
                       m:表示多行模式，即在到达一行文本末尾时还会继续查找下一行中是否存在与模式匹配的项。
